# Psychic-Letter-Game

Guess the alphabetic character.

### How to play?

- You have 9 chances to guess a alphabetic character!
    - Make sure don't repeat your guess.
    - Make sure all characters are in between a to z.

### Built with

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Github for versioning
