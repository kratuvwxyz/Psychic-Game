# Psychic-Letter-Game

Guess the alphabetic character.

### How to play?

- You have to guess a alphabetic character.
- You have 9 chances to match that character!
    - Display in **Red** shows you how many chances left.
- Keep guessing alphabet letter until you find out or you miss last chance!
    - Make sure don't repeat your guess.
    - Make sure all characters are in between a to z.
- Once you guess the alphabet letter or not...
    - Computer generate new character randomly.
    - Wins or Losses add +1
    - Display in **Yellow** have your answer!

### Built with

- HTML5
- CSS3
- Bootstrap
- JavaScript
- Github for versioning
